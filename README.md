# AutoScriptShopBackend

## Datenbankwahl
Als Datenbank wird das in Plesk integrierte Datenbankhosting genutzt. Die Datenbank ist MariaDB. Der grund dafür ist dass wir uns dann keine gedanken mehr über das Hosting der Datenbank machen müssen.

## Schnittstellen

### Products Database
* Get all Products
* Set A Product(Name, Tags, Description, File)
* Get Product by Id
* Search for Products
* Create Product
* Filter Products
* get all products by Type
* PriceRange
* Freelancer Rating
* Language(python, eg.)

### User Database
* Check Authentication
* Get User Information
* Get new Password
* Set User Information
* Delete User

### Taskboard Database
* Get all Tasks
* Get Task by Id
* Search for Task
* Create Task
* Filter Tasks
